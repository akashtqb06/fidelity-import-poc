The CDN JIT engine compiles bg-[#e94560] and text-[clamp(...)] on demand. The compiled platform CSS does not. Elements below will be unstyled in the target.

What survives vs what doesn't

Standard Tailwind utilities are pre-compiled into the platform CSS. Arbitrary values require the CDN JIT to generate their rules dynamically.

bg-primary — in @theme
bg-background-dark — compiled
bg-zinc-700 — base palette

h-[calc(100vh-64px)] — arbitrary calc
text-[clamp(3rem,...)] — fluid type
